ABOUT ME

My career in user-experience design and engineering began in a psychology lab at the University of South Florida. Using Python to automate data processing in research piqued my interest in learning about the capabilities of code.

In 2019, I attended DigitalCrafts in Atlanta, Georgia, completing a 16-week intensive program in Full-Stack Web Development. With a specialization in front-end web development, I am passionate about user-interface and user-experience.

In addition to design and development, I enjoy performing in plays, film & tv and exploring Atlanta's running trails, which led me to build trailhead.app.

PROJECTS

Matchgame is a React Native mobile app for creating sports and leisure pickup games. Users can select favorite games, create teams and compete for bragging rights in their area.

User interface designed using: Sketch, CSS and Native Base

Development tools: React Native, NodeJS, PostGreSQL

Demo Video GitHub

Trailhead is a trail running mobile app for finding running paths in Georgia and logging run activity. Users can search trails by city, save routes for future trips and record run dates.

Built using: React, Trail Run Project API and Styled Components.

Live Site GitHub

Jeopardy.app allows users to select and play Jeopardy episodes from an archive of past episodes and over 200,000 questions. The application backend logs episodes played, game statistics and ranks players on a leaderboard.

Built using: JavaScript, Express, Postgres and Bootstrap.

Live Site GitHub

myDoppel is a facematching web app that lets users find their character doppelganger from a selection of popular TV shows. The app utilizes faceAPi and TV-Maze API to run rapid facial comparison.

Built using: JavaScript, API fetching and DOM Manipulation.
